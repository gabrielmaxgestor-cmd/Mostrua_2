# Mostrua — Catálogo Online

Mostrua é uma plataforma para revendedores criarem sua loja online e venderem via WhatsApp.

## Funcionalidades planejadas (pós-MVP)
- Integração com IA (Google Gemini / @google/genai):
  - Sugestão automática de descrição de produtos
  - Análise de melhores horários para postar no WhatsApp
  - Estas features podem ser ativadas adicionando `@google/genai` de volta quando necessário.
